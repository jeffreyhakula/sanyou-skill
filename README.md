# 叁侑食堂 Claude Code Skill

叁侑食堂的 AI 助手 Skill，安装后可以直接问地址、营业时间、菜品、外卖等问题。

## 功能

- 餐厅地址与营业时间查询
- 菜品特色介绍（卤肉饭、牛肉饭、特色馄饨、中式糖水）
- 美团外卖订餐引导

## 安装

**Claude Code**
```bash
git clone https://github.com/jeffreyhakula/sanyou-skill ~/.claude/skills/sanyou-skill
```

**Cursor / Windsurf / Trae**
```bash
git clone https://github.com/jeffreyhakula/sanyou-skill ~/Documents/skills/sanyou-skill
```

安装后重启 IDE 即可生效。

## 使用示例

- "叁侑食堂在哪？"
- "几点开门？"
- "有什么好吃的？"
- "能点外卖吗？"

## 关于叁侑食堂

合肥市科学大道华地紫园小区，主营卤肉饭、牛肉饭、特色馄饨、中式糖水。营业时间 10:30 - 20:30。
